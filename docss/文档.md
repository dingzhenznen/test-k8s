## 一、KEP 总入口（Kubernetes Enhancement Proposal，官方功能设计原稿）
KEP 是 Kubernetes **最权威底层设计文档**，所有控制器、API、组件最初设计思路都在这里
1. KEP 官网列表页：https://www.kubernetes.dev/resources/keps/
2. KEP GitHub 源码仓库（所有设计文档原文）：https://github.com/kubernetes/enhancements
3. 快速访问单个KEP简写域名：`https://kep.k8s.io/KEP编号`
